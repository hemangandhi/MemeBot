# MemeBot
A Discord bot that sends memes from Reddit.

Made in Python, using [PRAW](https://github.com/praw-dev/praw), [Discord.py](https://github.com/Rapptz/discord.py), [dotenv](https://github.com/theskumar/python-dotenv)
